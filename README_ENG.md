TEKO Visualizer

TEKO Visualizer is a desktop application designed for working with document cameras in educational, presentation, and demonstration environments. The program provides stable image capture from IPEVO USB cameras, real-time video display, and quick saving of materials as images or documents.

Main features:

📷 Connection of document cameras via USB (UVC)

🎥 Real-time video streaming

⚡ Optimized performance for high resolutions

🔎 Selection of available camera resolutions

📊 Display of current FPS

🖼 Image capture

📄 Saving captured materials

🔄 Automatic camera detection

Features:

Works with IPEVO document cameras. Written in C++ / Qt. Uses OpenCV for image processing. Designed to run on Astra Linux 1.7 / 1.8.

Technologies:

C++
Qt
OpenCV
V4L2 (Video4Linux)

Supported platforms:

Astra Linux
Other Linux distributions with V4L2 support

This repository contains ready-to-use installation packages for TEKO Visualizer.

The source code of the program is not published in this repository.

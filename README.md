TEKO Visualizer
TEKO Visualizer — это настольное приложение для работы с документ-камерами в образовательных, презентационных и демонстрационных системах. Программа предназначена для стабильного захвата изображения с USB-камер IPEVO, отображения видеопотока в реальном времени и быстрого сохранения материалов в виде изображений или документов.

Основные возможности:

📷 Подключение документ-камер через USB (UVC)

🎥 Отображение видеопотока в реальном времени

⚡ Оптимизированная работа с высокими разрешениями

🔎 Выбор доступных разрешений камеры

📊 Отображение текущего FPS

🖼 Захват изображений

📄 Сохранение материалов

🔄 Автоматическое обнаружение камер

Особенности:
Работает с документ-камерами IPEVO , написан на C++ / Qt. Использует OpenCV для обработки изображений. Работает в средах Astra Linux 1.7 / 1.8. 

Технологии:
C++
Qt
OpenCV
V4L2 (Video4Linux)

Поддерживаемые платформы:
Astra Linux 
Другие Linux-дистрибутивы с поддержкой V4L2

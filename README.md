TEKO Visualizer
TEKO Visualizer — это настольное приложение для работы с документ-камерами в образовательных, презентационных и демонстрационных системах. Программа предназначена для стабильного захвата изображения с USB-камер (UVC), отображения видеопотока в реальном времени и быстрого сохранения материалов в виде изображений или документов.
Основная цель проекта — предоставить простое, надёжное и производительное решение для работы с документ-камерами в средах Linux (в первую очередь Astra Linux) без использования проприетарного ПО производителей.

Основные возможности:
📷 Подключение документ-камер через USB (UVC)
🎥 Отображение видеопотока в реальном времени
⚡ Оптимизированная работа с высокими разрешениями
🔎 Выбор доступных разрешений камеры
📊 Отображение текущего FPS
🖼 Захват изображений
📄 Сохранение материалов
🔄 Автоматическое обнаружение камер
📝 Логирование работы программы

Особенности:
Ориентирован на документ-камеры IPEVO и аналогичные UVC-устройства, написан на C++ / Qt. Использует OpenCV для обработки изображений. Работает в средах Astra Linux 1.7 / 1.8. Не требует фирменных драйверов производителя

Типичные сценарии использования:

Образование и дистанционное обучение
Демонстрация документов и предметов на презентациях
Визуализация печатных материалов
Использование в конференц-системах
Работа с интерактивными классами

Технологии:
C++
Qt
OpenCV
V4L2 (Video4Linux)

Поддерживаемые платформы:
